## 获取图片资源的方法

### 方法 1：使用 Unicode 音乐符号（推荐）

在 Markdown 中使用 Unicode 音乐符号代替部分图片：

**音符符号**：
- 全音符：𝅝 (U+1D15D)
- 二分音符：𝅗𝅥 (U+1D15E)
- 四分音符：♩ (U+2669) 或 𝅘𝅥 (U+1D15F)
- 八分音符：♪ (U+266A) 或 𝅘𝅥𝅮 (U+1D160)

**谱号符号**：
- 高音谱号：𝄞 (U+1D11E)
- 低音谱号：𝄢 (U+1D122)

**变音记号**：
- 升号：♯ (U+266F)
- 降号：♭ (U+266D)
- 还原号：♮ (U+266E)

**力度记号**：
直接使用文本 `pp`, `p`, `mf`, `f`, `ff` 即可

### 方法 2：使用免费图标库

**推荐资源**：

1. **Flaticon** (https://www.flaticon.com)
   - 搜索 "music notation", "piano", "staff"
   - 选择 CC BY 或免费许可的图标
   - 下载 PNG 格式（推荐 512x512 或更大）

2. **Font Awesome** (https://fontawesome.com)
   - 提供音乐相关的矢量图标
   - 可以导出为 PNG

3. **Noun Project** (https://thenounproject.com)
   - 搜索 "musical notation"
   - 选择免费或 CC0 许可的图标

4. **Wikimedia Commons** (https://commons.wikimedia.org)
   - 搜索 "musical notation", "piano keyboard"
   - 所有资源都是自由许可的

### 方法 3：使用 Flutter SVG

在 `pubspec.yaml` 添加：

```yaml
dependencies:
  flutter_svg: ^2.0.7
```

将 SVG 文件放在 `assets/images/` 下，然后在代码中使用：

```dart
import 'package:flutter_svg/flutter_svg.dart';

SvgPicture.asset(
  'assets/images/staff/treble_clef.svg',
  width: 60,
  height: 80,
)
```

### 方法 4：自己生成简单图片

使用 Python PIL/Pillow 生成简单的音符图片：

```python
from PIL import Image, ImageDraw, ImageFont

def create_staff_image():
    img = Image.new('RGB', (400, 200), 'white')
    draw = ImageDraw.Draw(img)
    
    # 画五条线
    for i in range(5):
        y = 40 + i * 30
        draw.line([(20, y), (380, y)], fill='black', width=2)
    
    img.save('five_lines.png')

create_staff_image()
```

## 临时解决方案

如果暂时没有图片资源，可以：

1. **使用 emoji 和 Unicode 符号**
   ```markdown
   > 🎵 高音谱号形状像优美的花体字母 G
   ```

2. **使用文字描述**
   ```markdown
   > 💡 想象：全音符就像一个空心的鸡蛋，二分音符是鸡蛋长了一根棍子
   ```

3. **使用占位符**
   ```markdown
   [图片：五线谱示意图]
   ```

## 版权注意事项

- ✅ 使用 CC0、CC BY 许可的资源
- ✅ 使用公共域（Public Domain）的资源
- ✅ 自己创作的图片
- ❌ 不要使用受版权保护的教材截图
- ❌ 不要使用未授权的商业图片

## 推荐做法

对于教育应用，建议：

1. 使用 Unicode 音乐符号替代简单的音符图片
2. 重要概念图使用自己绘制或免费图标库
3. 添加详细的文字说明作为补充
4. 未来可以逐步完善图片资源
