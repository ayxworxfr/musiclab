## 📝 数据格式

每个乐谱文件都包含以下信息：
- `id`: 唯一标识符
- `title`: 曲目名称
- `difficulty`: 难度等级 (1-5)
- `category`: 分类（folk, classical, pop, exercise, children）
- `metadata`: 元数据（作曲家、调号、拍号、速度等）
- `measures`: 小节数据（音符、歌词、指法等）
- `isBuiltIn`: 是否为内置乐谱

## 🔧 技术说明

### 加载机制
1. 应用启动时从 `sheets_index.json` 读取乐谱索引
2. 按需加载各个乐谱的 JSON 文件
3. 如果加载失败，会回退到硬编码的示例数据

### 音符表示
- **度数 (degree)**: 1-7 表示 do-si
- **八度 (octave)**: 0 表示中音区，1 表示高八度，-1 表示低八度
- **时值 (duration)**: whole（全音符）、half（二分音符）、quarter（四分音符）、eighth（八分音符）、sixteenth（十六分音符）
- **附点 (isDotted)**: true 表示附点音符
- **升降号 (accidental)**: sharp（升号）、flat（降号）、none（无）
- **歌词 (lyric)**: 可选的歌词字段
- **指法 (fingering)**: 可选的指法建议（1-5）

### 简谱格式
应用还支持简谱文本格式的导入，示例：
```
标题：小星星
作曲：莫扎特
调号：C
拍号：4/4
速度：100

1 1 5 5 | 6 6 5 - |
一 闪 一 闪 | 亮 晶 晶 |

4 4 3 3 | 2 2 1 - |
满 天 都 是 | 小 星 星 |
```
